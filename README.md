# Productivity Radar

Современный веб-сервис для отслеживания привычек, анализа продуктивности и визуализации личной статистики.

---

## О проекте

**Productivity Radar** — это персональный трекер привычек и продуктивности, позволяющий отслеживать выполнение ежедневных задач, анализировать уровень энергии и получать статистику активности.

Проект сочетает в себе современный веб-интерфейс, REST API и систему аналитики, позволяющую оценивать эффективность привычек в долгосрочной перспективе.

---

## Возможности

* Создание и управление привычками
* Отметка выполнения привычек
* Оценка уровня энергии
* Аналитика за неделю
* Статистика продуктивности
* Процент выполнения привычек
* Определение наиболее продуктивного дня недели
* REST API
* Адаптивный интерфейс
* Темная тема

## Основные сущности

### Habit

Привычка пользователя:

* название
* цвет
* дата создания

### Entry

Запись выполнения привычки:

* дата
* выполнена или нет
* уровень энергии
* связь с привычкой

---

## REST API

### Получить список привычек

```http
GET /api/habits/
```

### Создать привычку

```http
POST /api/habits/create/
```

### Получить статистику

```http
GET /api/stats/
```

### Получить записи

```http
GET /api/entries/
```

---

## Аналитика

Система рассчитывает:

* процент выполнения привычек;
* средний уровень энергии;
* наиболее продуктивный день недели;
* недельную статистику;
* динамику выполнения задач.
